This is a simple desktop weather application built using Python and Tkinter.  
It allows users to enter a city name and view real-time weather information such as temperature, humidity, wind speed, and weather condition.

## Features
- Simple and clean GUI using Tkinter
- Real-time weather data
- Displays temperature in Celsius
- Shows humidity, wind speed, and condition
- Error handling for invalid city and no internet

## Technologies Used
- Python
- Tkinter
- Requests library
- OpenWeatherMap API
